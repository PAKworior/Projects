# Chess Project

## Overview

This project is still under development. Currently, I am working on integrating network features to enable online play and ensuring network security.

## How to Run

### Requirements

- **Operating System:** Windows
- **Compiler:** GCC compiler must be installed and properly configured (or just save your braincells, and run the "chess_code.cpp" in visual studio, this will work fine)

To verify your GCC installation, run:
```bash
gcc --version
```

### Setup Instructions
you can skip all these steps, and just run the code in "visual studio"
1. **Save the project** to a location on your computer and copy the file path.

2. **Open Windows Command Prompt** and navigate to the project directory:
   ```bash
   cd {file_path}
   ```
   *Replace `{file_path}` with the actual path to your project folder.*

3. **Compile the project** using the G++ compiler:
   ```bash
   g++ Chess_Project.cpp -o Chess_Project.exe
   ```

4. **Run the executable:**
   ```bash
   ./Chess_Project.exe
   ```

The game will now launch in your terminal.

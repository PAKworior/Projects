## how to run it
 1: my code is only compatible on windows for now
 2: you must have the gcc compiler installed and running(use "gcc --version" to ensure that)
 3: save the code somewhere in the computer and copy the file path
 4: go to wondows command prompt, and run "cd {file path} " in case you only have 2 brain cells, you are suppose to replace " {file path} " with the actual link of the copied foldier
 5: run the command "g++ Chess_Project.cpp -o Chess_Project.exe" 
 6: finally, run "./program.exe" in terminal, and the code will run
